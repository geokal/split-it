# Documentation

This folder contains organized documentation for the Split-It refactoring project.

## Structure

### `/components/`
Component-related documentation:
- `COMPONENT_DEPENDENCIES.md` - Mapping of component dependencies
- `EXTRACTION_STANDARD.md` - Standard approach for component extraction
- `PARAMETER_CONTRACTS.md` - Parameter contracts for components
- `SPLITTING_GUIDE.md` - Guide for splitting markup and code-behind

### `/wiring/`
Component wiring documentation:
- `WIRING_PROGRESS.md` - Current wiring progress and status
- `WIRING_PLAN_*.md` - Detailed wiring plans for specific components
- `WIRING_VERIFICATION_*.md` - Verification reports for wired components
- `WIRING_APPROACH_COMPARISON.md` - Comparison of wiring approaches
- `WIRING_PATTERN_EXPLANATION.md` - Explanation of wiring patterns
- `WIRING_TEST_PLAN.md` - Plan for testing wiring functionality

## Main Documentation

For the main project overview, see:
- `../AGENTS.md` - Main project document with tasks and guidelines
- `../PROGRESS.md` - Progress tracking
- `../REFACTORING_PLAN.md` - Strategic refactoring plan
